本项目针对青少年的兴趣爱好，采用本团队研发的材料包，开发一款基于安卓的在线课堂APP。通过开发的APP可实现查看课程介绍、在线课程购买、材料包购买、指导视频、上传作品、教师评价、活动推广、积分会员功能、意见反馈等。
